# プロダクト開発 実践編

## ポジネガ日記

こちらのアプリケーションでは、1日に良いことを3つ記載する従来の日記アプリに加えて、
日記を記載する前に、数分設けることで、その時間内に、ひたすらに書きたいことを書き出す（デトックス）機能を追加する
ことで、嫌なことは忘れてスッキリした状態で、いいことを振り返れるようにするアプリケーションである。
